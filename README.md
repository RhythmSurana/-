# 🎥 YouTube Video Chatbot (RAG + LangChain)

Turn any YouTube video into a chatbot.

Paste a video link, ask questions, and get instant, relevant answers — without watching the entire video.

---

## 💡 Overview

We all spend hours watching long YouTube videos just to find one small piece of information.

This project solves that.

It allows users to:
- Input a YouTube video link
- Convert the video into a searchable knowledge base
- Ask questions in natural language
- Get precise answers instantly

---

## ⚙️ How It Works (Simple Explanation)

1. The YouTube video is converted into text (transcript)
2. The text is split into smaller chunks
3. Relevant chunks are retrieved based on your question (**RAG**)
4. A language model generates a contextual answer

---

## 🧠 Tech Stack

- **LangChain** – for building the RAG pipeline  
- **OpenAI / LLM** – for generating answers  
- **FAISS / Vector DB** – for storing embeddings  
- **Streamlit** – for UI  
- **YouTube Transcript API** – for extracting video text  

---

## ✨ Features

- 🔗 Paste any YouTube video link  
- 💬 Chat with the video content  
- ⚡ Instant answers  
- 🎯 Context-aware responses  
- 🧠 Uses Retrieval-Augmented Generation (RAG)  
- 🌐 Deployed on Streamlit Cloud
